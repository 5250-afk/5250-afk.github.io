Progress

  • Day 1: Basic structure

    • Basic structure can be added using tags like <html>, <head>, <body>, <header>, <main>, <footer>, and <p>.

    <html> is the root element for the html page

    <head> contains metadata

    <body> contains all visible content

    <header> can contain items like titles, logos, and links

    <main> holds the primary content of the page

    <footer> adds content to the bottom of the page like copyright info

    <p> starts a new paragraph

  • Day 2: Adding links and images

    • Images can be added using <img src="image-name-and-path" alt="alt- text" width="size">

    Links can be added using <a href="URK" target="_blank">Link Name</a>
